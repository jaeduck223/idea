- 발화자의 목소리의 높낮이나 억양에 따른 상황의 긴급도 분석 서비스(in 병원, 사고 현장에서의 도움 요청)

LocalLLM을 이용하여 키오스크 이용에 어려움을 겪는 디지털 소외 계층을 위한 채팅을 기반으로 힌 주문 보조 서비스
- 실제 식당이나 상점에서의 사용 실험을 진행하지 못한 것이 아쉬움.
